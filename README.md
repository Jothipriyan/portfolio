## ğŸ¨ Customization Guide

### Personal Information
Edit the following sections in `index.html`:

- **Name and Title**: Update the hero section
- **Contact Information**: Update email, phone, and location
- **About Me**: Update the description and statistics
- **Education**: Update educational background
- **Experience**: Add your work experience

### Colors and Styling
Modify `styles.css` to change:

- **Fonts**: Change the Google Fonts import and font-family declarations
- **Spacing**: Adjust margins and padding values

2. Add corresponding CSS styles in `styles.css`
3. Update the navigation menu to include the new section
4. Add smooth scrolling functionality in `script.js`
